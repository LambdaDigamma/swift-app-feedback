## Installation

Install `AppFeedback` with Swift Package Manager:

```swift
dependencies: [
    .package(name: "AppFeedback", url: "https://github.com/lambdadigamma/swift-app-feedback", .upToNextMajor(from: "1.0.0")),
]
```
